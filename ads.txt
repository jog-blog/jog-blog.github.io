# Google AdSense ads.txt文件
# 该文件用于验证您是网站的合法所有者并帮助防止广告欺诈
# 请将以下行中的发布商ID替换为您自己的Google AdSense发布商ID

# Google AdSense条目格式:
# google.com, pub-您的发布商ID, DIRECT, f08c47fec0942fa0

# 示例（请替换为您自己的发布商ID）:
# google.com, pub-1234567890123456, DIRECT, f08c47fec0942fa0

# 使用说明:
# 1. 登录您的Google AdSense账户
# 2. 点击左侧菜单中的"网站"选项
# 3. 找到"ads.txt"部分，复制完整的ads.txt内容
# 4. 将复制的内容粘贴到这个文件中，替换上面的示例内容
# 5. 保存文件后运行"hugo"命令重新构建网站

# 注意事项:
# - ads.txt文件必须放置在网站的根目录下（通过static目录实现）
# - 文件名必须是小写的"ads.txt"，不能有其他扩展名
# - 文件必须是纯文本格式
# - Google AdSense可能需要24-48小时来验证您的ads.txt文件
